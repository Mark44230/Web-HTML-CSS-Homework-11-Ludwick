    The purpose of this site is to visualize the weather of over 500 selected cities across the world with, of course, varying distances from the equator. To create the visualizations, the data was pulled from OpenWeatherMap API and plotted using Matplotlib. Scatterplots were built to showcase the following relationships: * Temperature (F) vs. Latitude * Humidity (%) vs. Latitude * Cloudiness (%) vs. Latitude * Wind Speed (mph) vs. Latitude.

    These four plots and explanations of each can be accessed from this page by clicking on the pictures below or by using the drop down menu. The site also includes a comparison of the four charts and the data used to create the charts.

    Several cautions to be used in the interpretation of the data; these cautions are noted in the chart explanations.
